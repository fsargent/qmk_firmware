# Charybdis "lite" (4x6)

An ergonomic keyboard with integrated trackball.

This is a fork of
[`keyboards/bastardkb/charybdis/4x6`](../../charybdis/4x6/readme.md) with
trimmed down features and RGB Matrix and Via/Vial support.
